### En:
#### Task 3
Task: Extend the Delivery List Application
Enhance your delivery list mini-application by adding a static method to the EditDelivery class that calculates the total distance of all deliveries.
The method should accept a single argument — an array of delivery class instances.

Example of Method Usage
const totalDistance = EditDelivery.getTotalDistance(deliveryArr);

Important: Deliveries with the status "canceled" should not be included in the total.

Additional Requirements
Add a button to the application that, when clicked, calls the total distance calculation method and displays the result on the screen.

Evaluation Criteria
- No errors in the console
- The application code works correctly and meets the task requirements
- JavaScript modules and classes are used
- DOM elements are styled appropriately

### Ru:
#### Задача 3
Доработайте мини-приложение списка доставки и добавьте в класс EditDelivery статический метод расчёта суммарного расстояния всех доставок. У метода должен быть единственный атрибут, являющийся массивом классов доставок.

Пример вызова метода для расчёта общего расстояния:

сonst totalDistance = EditDelivery.getTotalDistance(deliveryArr)

Важно то, что доставки со статусом canceled (отменена) суммироваться не должны.

Добавьте в приложении кнопку, при клике на которую будет вызван метод расчёта общего расстояния, и покажите итоговый результат на экране

Код приложения работает корректно, в соответствии с заданием.
В работе используются модули с классами.
